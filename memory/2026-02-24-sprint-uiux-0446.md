## Critical Issues (P0)

### P0-001: Cash Flow Chart Missing Loading Skeleton
**Problem:** Chart canvas shows empty white space for 0.5-2s while Chart.js loads from CDN  
**Impact:** Users perceive broken UI  
**Fix:** Add `.chart-skeleton-overlay` wrapper (infrastructure already exists in components.css line 1191)  
**Effort:** 1 hour

### P0-002: Bills Aging Buckets Missing Focus States
**Problem:** Keyboard users can't see which bucket is focused (WCAG 2.4.7 violation)  
**Impact:** Accessibility failure  
**Fix:** Add `.bills-bucket:focus-visible` CSS rule with blue ring + box-shadow  
**Effort:** 30 min
