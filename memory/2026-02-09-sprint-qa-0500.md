## Bugs Found

### FC-NEW-001 (P1 - FIXED ✅)
**Problem:** Transactions page empty state not showing when no data  
**Cause:** Race condition - `getElementById` check ran before DOM ready  
**Fix:** Moved check inside `DOMContentLoaded` listener  
**Commit:** d1bbd85  
**Files:** app/assets/js/transactions.js (lines 297-307)  
**Status:** Fixed, committed, pushed, awaiting Azure deployment

### FC-NEW-002 (P2 - OPEN)
**Problem:** 61 hardcoded hex colors in main.css violate design tokens  
**Impact:** Hard to maintain, breaks theme consistency  
**Examples:** `#fff`, `#1a1a1a`, `#dee2e6` should use CSS variables  
**Fix Required:** Systematic find/replace with design token variables  
**Files:** app/assets/css/main.css  
**Issue:** https://github.com/Fireside-Cloud-Solutions/fireside-capital-dashboard/issues/2
