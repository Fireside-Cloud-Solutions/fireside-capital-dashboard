## Bugs Fixed

### FC-UIUX-018 — Reports Page Export Button Hierarchy (P2, 2 min)

**Source:** Sprint UI/UX Audit (Session 0645, 2026-02-16)  
**File:** `app/reports.html` line 80

**Problem:**
- Export button used `btn btn-secondary` (blue filled)
- Should use `btn btn-outline-secondary` for utility actions
- Violated button hierarchy system (primary = orange CTAs, secondary = page actions, outline = utilities)

**Fix:**
```html
<!-- BEFORE -->
<button class="btn btn-secondary" aria-label="Export financial report as CSV">
  <i class="bi bi-download"></i> Export
</button>

<!-- AFTER -->
<button class="btn btn-outline-secondary" aria-label="Export financial report as CSV">
  <i class="bi bi-download"></i> Export
</button>
```

**Impact:**
- ✅ Button hierarchy 100% consistent across all pages
- ✅ Export correctly styled as utility action (outline)
- ✅ Visual consistency improved

### FC-UIUX-019 — Reports Page Section Spacing Inconsistency (P2, 5 min)

**Source:** Sprint UI/UX Audit (Session 0645, 2026-02-16)  
**File:** `app/reports.html` lines 160, 188

**Problem:**
- Chart rows used `mt-3` (16px margin-top, Bootstrap default)
- Breaks design system's 8px grid (should be multiples of 8)
- Inconsistent spacing across page sections

**Fix:**
```html
<!-- BEFORE -->
<div class="row g-3 g-xl-4 mt-3">
  <!-- Additional Charts (Moved from Dashboard) -->
</div>

<div class="row g-3 g-xl-4 mt-3">
  <!-- Savings Rate Over Time -->
</div>

<!-- AFTER -->
<div class="row g-3 g-xl-4 mb-24">
  <!-- Additional Charts (Moved from Dashboard) -->
</div>

<div class="row g-3 g-xl-4 mb-24">
  <!-- Savings Rate Over Time -->
</div>
```

**Changes:**
- `mt-3` (16px) → `mb-24` (24px = 3 × 8px)
- Changed margin-top to margin-bottom (proper spacing flow)
- Now matches design system's 8px grid

**Impact:**
- ✅ Spacing matches 8px grid system (24px = 3 × 8px)
- ✅ Consistent with other page sections
- ✅ Visual rhythm improved
