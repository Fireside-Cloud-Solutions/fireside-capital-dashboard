## Investigation: Budget Page

**Finding:** Budget page (line 105) intentionally uses `btn-sm` for "Add Budget Item" button.

**Rationale:**  
Budget page header is a toolbar with 5 controls:
1. Previous Month button (btn-outline-secondary)
2. Current Month display (h4)
3. Next Month button (btn-outline-secondary)
4. Generate Budget button (btn-secondary btn-sm)
5. Add Budget Item button (btn-primary btn-sm)

**Decision:** NOT a bug — toolbar-style headers use smaller buttons to fit multiple controls. This is a valid design pattern.
