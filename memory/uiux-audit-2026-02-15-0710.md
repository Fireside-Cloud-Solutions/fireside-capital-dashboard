## Issues Found (6 total)

### P0 CRITICAL (2 issues)

**1. BUG-UI-NAV-001: Mobile Navigation Z-index Conflict**
- **Location:** `index.html` inline styles (line ~41)
- **Problem:** Hamburger menu uses `z-index: var(--z-modal)` (400) which conflicts with modal overlays
- **Impact:** Breaks mobile navigation when modals are open
- **Fix:** Change to `z-index: var(--z-sticky)` (200)
- **Effort:** 5 minutes

**2. BUG-UI-CSS-001: Duplicate Inline Critical CSS**
- **Location:** All 11 HTML files have identical 40-line `<style>` block in `<head>`
- **Problem:** Maintenance nightmare, violates DRY principle, changes require 11-file updates
- **Impact:** Code maintainability, future update complexity
- **Fix:** Extract to `assets/css/critical.css`, link from all pages
- **Effort:** 10 minutes

### P1 HIGH (2 issues)

**3. BUG-UI-NOTIF-001: Notification Dropdown Excessive Width**
- **Location:** `components.css` line ~73
- **Problem:** Dropdown is 550px wide (unusually large, causes tablet horizontal scroll)
- **Impact:** Poor UX on tablets/small desktops
- **Fix:** Reduce to 380px (Bootstrap convention)
- **Effort:** 2 minutes

**4. BUG-UI-TYPE-001: Inconsistent Typography Units**
- **Location:** Throughout `main.css` and `components.css`
- **Problem:** Mix of px and rem values breaks relative scaling
- **Impact:** Accessibility (zoom), responsive typography
- **Fix:** Replace hardcoded px with design token variables
- **Effort:** 30 minutes (find-replace across CSS files)

### P2 MEDIUM (2 issues)

**5. BUG-UI-LAYOUT-001: Page Header Layout Inconsistency**
- **Location:** `assets.html`, `bills.html`, etc. (vs `index.html`)
- **Problem:** 3-div structure breaks `justify-content: space-between` layout
- **Impact:** Visual inconsistency, awkward spacing
- **Fix:** Wrap actions + auth in single container
- **Effort:** 15 minutes (update 10 HTML files)

**6. BUG-UI-BTN-001: Button Action Hierarchy Violations**
- **Location:** `transactions.html`, `bills.html`, and other pages
- **Problem:** Multiple adjacent buttons use `btn-secondary` (blue), violating tri-color hierarchy
- **Impact:** Poor visual hierarchy, confusing UX
- **Fix:** Apply proper hierarchy (Orange PRIMARY, Blue SECONDARY, Gray TERTIARY)
- **Effort:** 30 minutes (review all 11 pages)
