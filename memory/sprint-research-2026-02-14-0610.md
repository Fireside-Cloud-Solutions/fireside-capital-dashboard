### 3. Service Worker (FC-108) ‚Äî 3-4h, Offline Support

```javascript
// sw.js ‚Äî Hybrid caching strategy
const CACHE_VERSION = 'fireside-v1';

self.addEventListener('fetch', (e) => {
  const { request } = e;
  
  // API requests: Network-first with cache fallback
  if (request.url.includes('/rest/v1/')) {
    e.respondWith(
      fetch(request)
        .then(res => {
          caches.open(CACHE_VERSION).then(c => c.put(request, res.clone()));
          return res;
        })
        .catch(() => caches.match(request))
    );
  }
  // Static assets: Cache-first (instant loading)
  else {
    e.respondWith(
      caches.match(request).then(cached => cached || fetch(request))
    );
  }
});
```

**Impact:** App works offline, instant loading (< 100ms for cached resources).

```html
<!-- Inline script in <head> to prevent FOUC -->
<script>
  (function() {
    const theme = localStorage.getItem('theme') || 
      (window.matchMedia('(prefers-color-scheme: dark)').matches ? 'dark' : 'light');
    document.documentElement.setAttribute('data-bs-theme', theme);
  })();
</script>
```
