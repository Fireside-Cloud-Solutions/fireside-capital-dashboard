**File:** `app/assets/css/responsive.css`  
**Problem:** 107 !important declarations (37% of all !important usage across entire codebase)

**Impact:**
- Makes CSS hard to override/maintain
- Indicates specificity wars
- Violates CSS best practices
- Increases technical debt

**Recommendation:**
1. Refactor responsive.css to use proper specificity instead of !important
