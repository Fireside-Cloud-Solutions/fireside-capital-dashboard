**Recommended Fix:**

**Option 1: Conditional Logging (Quick Fix - 2h)**
```javascript

  debug: (...args) => { if (window.DEBUG_MODE) console.log(...args); }
};

// Replace all console.log with logger.log
```

**Option 2: Build-Time Stripping (Proper Solution - 3h)**
- Use Terser or Rollup plugin to strip console.log in production builds
- Keep console.warn and console.error for actual issues
- Add --debug flag for development builds
